# StacyR-art.github.io
### Artist Statement
###### Making art is a process that I enjoy and use to connect with various of things. It can be communities, my emotions, my experiences, etc. I chose this career path and interdisciplinary program because I know I would be able to tell the narratives I want through new technical and analytical skills. My practice is very important to me and I hope to continue to share more.
